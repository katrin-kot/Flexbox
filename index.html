What is Flexbox?

Flexible Box Layout Module

A Flexbox consists of:

  • a flex container — the parent container
  • flex elements — child blocks
A Flexbox consists of: a flex-container-the parent container, flex elements - child blocks

The flexbox module allows you to solve the following tasks:

  • Position elements in one of four directions;
  • Redefine the order in which items are displayed;
  • Automatically dimension the elements so that they fit into the available space;
  • Solve the problem with horizontal and vertical centering;
  • Carry items inside the container, preventing them from overflowing;
  • Create columns of the same height;
  • Create pressed to the bottom of the page;

Properties for the Parent


              .flex-container { 
               display: flex; 
              }
To make the block flex you need to add a property to the parent container: display:flex

The flex container properties are:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content
This is other properties.

              .container {
                flex-direction: row | row-reverse | column | column-reverse;
              }
This property arranges items in a row or in a column.

              .container {
                flex-wrap: nowrap | wrap | wrap-reverse;
              }
nowrap (default): all flex items will be on one line wrap: flex items will wrap onto multiple lines, from top to bottom. wrap-reverse: flex items will wrap onto multiple lines from bottom to top.

flex-flow


            .container{
            flex-flow: <‘flex-direction’> || <‘flex-wrap’>
          }
This is a shorthand flex-direction and flex-wrap properties

              .container {
                justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
              }
This property positions elements horizontally.

              .container {
                align-items: flex-start | flex-end | center | baseline | stretch;
              }
This property positions elements vertically.

            .container {
              align-content: flex-start | flex-end | center | space-between | space-around | stretch;
            }

Properties for the Children


                .item {
                  order: ; /* default is 0 */
                }
The order property controls the order in which they appear in the flex container.

                .item {
                  flex-grow: ; /* default is 0 */
                }
It dictates what amount of the available space inside the flex container the item should take up. 

                    .item {
                      flex-shrink: ; /* default 1 */
                    }
This defines the ability for a flex item to shrink if necessary.

This defines the default size of an element before the remaining space is distributed.


                .item {
                  flex-basis:  | auto; /* default auto */
                }
This defines the default size of an element before the remaining space is distributed

            .item {
              flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]
            }
          

                .item {
                  align-self: auto | flex-start | flex-end | center | baseline | stretch;
                }
This allows the default alignment (or the one specified by align-items) to be overridden for individual flex items.
